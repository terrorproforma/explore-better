01 / Per-tab terminal

The shell is already in the right folder.

Every file tab can own a real Windows terminal. Open it only when needed, keep the process alive while switching tabs, and work without rebuilding paths by hand.

  • Real ConPTY sessionsPowerShell 7, Windows PowerShell, or Command Prompt rendered through xterm.js.
  • One session per file tabIndependent output, profile, selection, scroll position, and resizable drawer height.
  • Folder-aware, never intrusiveNew shells start in the tab folder and follow navigation only while the prompt is idle.
  • Narrow elevationOpen one administrator terminal through UAC without elevating the file manager or backend.

Ctrl + ` toggles the active pane terminal. Both pane terminals can stay open together.

02 / Local MCP AI Bridge

Give AI file tools, not a pile of terminal text.

Explore Better exposes its live panes, indexes, Analyzer, comparisons, and recoverable operation system as typed local MCP tools for Codex, Claude, Cursor, VS Code, and other stdio clients.

  • Read-first profilesAuthorize exact folders and tools per client; revoke a profile without changing the rest.
  • Structured contextActive tabs, visible selection, indexed discovery, checksums, duplicates, disk analysis, and jobs.
  • Preview before writesWrite profiles must plan first, then use a short-lived one-use token bound to current filesystem signatures.
  • Local security boundaryStdio sidecar, authenticated same-user named pipe, bounded messages, and no shell or terminal control.

07 / Operations

Fast should never mean reckless.

Copy, move, overwrite, and sync operations are staged, journaled, and recoverable after interruption.

  1. 01PlanPreview targets and conflicts
  2. 02StageBuild beside the destination
  3. 03CommitAtomically promote completed work
  4. 04RecoverResume, restore, or keep destination

Transactional operations

Sibling staging, backups, durable journal phases, cancellation, pause, resume, and deterministic recovery.

Search and index

Folder indexes, background content search, labels, saved searches, and responsive foreground work.

Workspace memory

Layouts, tab groups, pane snapshots, favorites, aliases, labels, selection sets, and file baskets.

Native Windows provider

A bundled normal-user Go helper handles fast Win32 listing and exact allocated-size lookup.

Desktop, not a web wrapper

Packaged Electron app, native shell entry points, auto-update feed, drag and drop, and Windows shortcuts.

Preview and compare

Built-in viewers, checksums, duplicate finding, folder comparison, labels, collections, and explicit sync plans keep decisions inspectable.

08 / Explorer integration

Make it the default. Change your mind later.

Explore Better can handle normal folder and drive opens, folder backgrounds, and file-location commands for your Windows account. No administrator access is required.

  • Exact backup of existing shell handlers
  • One-click Restore Previous action
  • Optional Win+E helper, off by default
  • Windows Explorer remains installed
